# Config

## List

* `$cios conf ls`

    ```shell
    ##############################################################################################
        Stage:          環境
        Client ID:      xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
        Client Secret:  xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
        Log Level:      info

    ###############################################################################################
    ```

## Store

* `$cios cf save`
    ⚠環境別に一つだけ保存します。

## Switch

* `$cios config switch`
    ⚠セーブした環境だけ切り替えが可能です。

    ```shell
    ? Choose a stage: zephyranthes
    Completed switch zephyranthes
    ```
