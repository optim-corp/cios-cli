# File Storage API

* [Bucket](./bucket.md)

    ```shell
    NAME:
        cios bucket - cios bucket | b

    USAGE:
        cios bucket command [command options] [arguments...]

    COMMANDS:
        create, cre, add, sub
        delete, del, remove, rm
        list, ls, show
        update, patch, up
        help, h                  Shows a list of commands or help for one command

    OPTIONS:
        --help, -h     show help (default: false)
        --version, -v  print the version (default: false)
    ```
